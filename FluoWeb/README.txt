Использование celery+redis+Flask.
redis - сетевое журналируемое хранилище данных типа "ключ-значение"
celery - асинхронная очередь6 основанная да обмене сообщениями
Flask - веб-фреймворк

Задачи управления устройствами выполняются при помощи celery.
Требовательные к времени выполнения задачи:
	 1. Перемещение спектрометра
	 2. Включение/выключение светодиодов, перемещение турели
	 3. Запуск АЦП на счет
	 4. Осуществление сбора данных по программе
